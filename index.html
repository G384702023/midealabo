「webコンテンツ作成」の成果

本テーマについて

この講義の目的はシラバシに書いてある通り、先行する情報工学の専門科目の基礎知識を応用して実践的な能力を修得することです。また、Webコンテンツ作成の目標は、HTML等を用いたWebページ作成ができるようになることです。

プロジェクト課題について

Webコンテンツ作成のプロジェクト課題では、世界の場所の天気や、気候の情報を検索できるWebサイトを作成しました。毎講義ごとにその講義で学んだことを活かして、地道に作成しました。今回のプロジェクト課題を作成したことは、講義で学んだことを実践として使用することができた良い機会だったと思います。また、学んだことを実践でも通じるという体験をできたのは、これからのさまざまな講義を受講していくにあたって、ものすごく良い経験だっったと思います。そういう点でもプロジェクト課題を行なっている時間は有意義だったと思います。

数当てゲーム

今回作成した数当てゲームは、機械が一つランダムで生成する1~10までの数字を自分の運や機械からのヒントをもとに、3回以内に言い当てるというゲームです。数当てゲームの作成で一番楽しかったことは、最初の方の講義ではまだ学んでいなくて、実装ができなかった機能を、その後の講義で実装方法を学び、機能を追加できたときです。例えば、第3回時点では予想した数は4に固定だったのに、最終的には自分で数を入力し予想することができた、などです。自分の成長や、その講義では具体的になにを学んだのかを理解することができてとても嬉しかったです。この経験を他の講義でも活かせるように努力をしていきたいと思います。

工夫点、苦心点、強調点、感想

Webコンテンツ作成での工夫点は、工夫点と言っていいか分かりませんが、講義中に理解しきれていなかったところを、後に講義資料など見返したり、友達に教わったりして、分からないままにせず、理解しきったことです。苦心点は、講義中に一回聞いただけのことや、講義資料を読んでもなかなか理解し難いところがあり、それを理解するためにかなりの労力を使ったことです。また、理解した通りにJava Scriptを記述しても、思った通りに動作してくれなかったことです。私はこのWebコンテンツ作成の講義を受講して、一番良かったと思ったことがあります。それは自分が講義で学んだことが本当に世の中の仕事へとつながっていると実感できたことです。というのもこの講義のWebコンテンツ作成では、第1回では WWW と HTML、第2回では CSS による Web ページの装飾、第3回では JavaScript の基本、第4回では DOM と JavaScript、第5回では イベント処理、第6回では 信号処理を学びました。これらの学んだことは、今までインターネットを使っていたらよく目にするものばかりです。それなので、間接的ではなく直接、世の中で使用されているということを学べたという経験をできたことがとても良かったです。他の情報系の講義でも、たくさん自分の知らないことを学んでいますが、まだいまいち将来のどのようなところに活きてくるのが分かっていないということがよくあります。それなのでこのWebコンテンツ作成の講義のおかげで、今大学で学んでいることが将来の自分に活かされるんだなということをとても実感できました。このことは私の大きな学びです。またこのWebコンテンツ作成の講義では、分からないことがあっても、自分が理解したいと思う気持ちがあれば、先生や先輩や友達に聞くことができたので、そういう環境で学ぶことができたということも良かった点です。また、演習を行う時間がかなり長く設定されていたので、時間のことを気にせず自分のペースで演習などを行うことができて良かったです。最後に、この講義では全く初めて知ることがほとんどでした。しかし、将来につながることや、今まで何気なく使用していたWeb機能などの性質など、自分の興味のあることを深く学べてとても有意義な講義だと思いました。この講義で学べたことを少しでも、他の講義や、これからの大学生活に活かせるよう努力していきたいと思います。
